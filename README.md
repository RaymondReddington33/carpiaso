# ASO Report Generator

A Next.js application for generating ASO (App Store Optimization) reports.

## Requirements / Requisitos

- Node.js 18+ 
- pnpm (package manager)

## Installation / Instalación

### English

1. Install dependencies:
```bash
pnpm install
```

2. Create a `.env.local` file in the root directory with your OpenAI API key:
```bash
OPENAI_API_KEY=your_openai_api_key_here
```
Get your API key from: https://platform.openai.com/api-keys

3. Run the development server:
```bash
pnpm dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

### Español

1. Instalar dependencias:
```bash
pnpm install
```

2. Crear un archivo `.env.local` en el directorio raíz con tu clave API de OpenAI:
```bash
OPENAI_API_KEY=tu_clave_api_aqui
```
Obtén tu clave API desde: https://platform.openai.com/api-keys

3. Ejecutar el servidor de desarrollo:
```bash
pnpm dev
```

4. Abrir [http://localhost:3000](http://localhost:3000) en tu navegador.

## Available Scripts / Scripts Disponibles

### English

- `pnpm dev` - Start development server
- `pnpm build` - Build for production
- `pnpm start` - Start production server
- `pnpm lint` - Run ESLint

### Español

- `pnpm dev` - Iniciar servidor de desarrollo
- `pnpm build` - Construir para producción
- `pnpm start` - Iniciar servidor de producción
